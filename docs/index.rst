Flask-Generic-Views
===================

.. module:: flask_generic_views

Flask-Generic-Views is an extension to `Flask`_ that provides a set of generic
class based views. It aims to simplify applications by providing a set of well
tested base classes and pluggable views for common tasks.

.. _Flask: http://flask.pocoo.org/
.. _example sourcecode:
   http://github.com/artisanofcode/python-flask-generic-views/tree/master/examples/
